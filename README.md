## Working With the Plugin

Insert EQ4 MS on a track or bus in your DAW. The interface offers two operating styles: standard stereo and mid-side. In standard mode, both channels are processed together. When you switch to mid-side mode, you can adjust the center (mid) and side components independently using the same EQ controls.

For mastering, begin with conservative moves. The air band shelf is ideal for adding clarity and sparkle without pushing the top end too hard. Use the low-frequency section to firm up the bass, and turn on the high-pass filter to clear away subsonic content. Mid-side mode is especially useful when you want to widen or narrow the stereo picture by adjusting side frequencies while leaving the center section unchanged.

Yes. The plugin runs with zero added latency, so it fits real-time monitoring, tracking, mixing, and mastering workflows.

**Is mid-side processing dependent on DAW routing?**  
No. Mid-side operation is handled inside the plugin itself. Just switch the mode on the interface and make your adjustments there.

**The plugin is missing from my DAW. What should I check?**  
First confirm the install finished properly, then rescan plugins from your DAW's manager. If it still does not show up, make sure your host supports the installed format, VST3 or AAX, and verify that the plugin folder is included in the scan paths.
